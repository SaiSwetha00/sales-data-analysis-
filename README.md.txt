# 🧮 Sales Data Analysis using Python


## 🎯 Objective
To analyze sales data and identify:
- Top-selling products
- Highest-performing regions
- Monthly sales trends


---


## 🧰 Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab


---


## 📊 Steps Performed
1. Loaded the dataset (`sales.csv`).
2. Cleaned and explored the data using Pandas.
3. Calculated total sales by product, region, and month.
4. Visualized sales trends using bar and line charts.


---


## 📈 Key Insights
- **Laptops** had the highest sales overall.
- **North region** performed consistently well.
- **March and May** showed strong sales growth.
